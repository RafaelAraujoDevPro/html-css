Listas Ordenadas

A HTML chama de ordered lists todas aquelas listas onde a ordem dos itens é algo muito importante, ou seja, onde há uma escala ordenada de importâcia dos itens.

Para criar uma ordered list, vamos usar a tag <ol> para delimitar a lista e <li> (list item) para identificar cada item da lista.

        
    <ol>
        <li>item 1</li>
        <li>item 2</li>
        <li>item 3</li>
        <li>item 4</li>
        <li>item 5</li>
    <ol>

 exemplo:

Lista afazeres de uma rotina diária

  1. Acordar
  2. Ligar pra Maria
  3. Tomar café
  4. Escovar os dentes
  5. Ir para a faculdade
  6. Almoçar
  7. Ir pro trabalho
  8. Voltar pra casa
  9. Jantar
  10. Dormir

Observação

 →IMPORTANTE! Segundo a W3C, a tag <ol> é de fechamento obrigatório (ou seja, devemos sempre usar </ol>). Já a tag <li> tem seu fechamento opcional a partir da HTML5.

A tag <ol> possui um parâmetro type, onde configuramos o tipo de marcador da lista atual. As opções de valores para esse parâmetro são:

  • 1 - valor padrão. Cria listas numeradas. Ex: 1, 2, 3, 4, 5...
  • A - Cria lista alfabética em maiúsculas. Ex: A, B, C, D, E...
  • a - Cria lista alfabética em minúsculas. EX: a, b, c, d, e...
  • I - Cria lista com algorismos romanos maiúsculas. Ex: I, II, III, IV, V...
  • i - Cria listas com algorismos romanos em minusculas. Ex: i, ii, iii, iv, v...

Você também pode indicar o inicio da contagem usando o parâmetro start.
Por exemplo, a tag <ol type="I" start = "5"> vai gerar itens numerados como V, VI, VII, VIII, IX, ...

 Listas não Ordenadas

As unordered list, também chamada de listas com marcadores, que são aquelas onde a ordem dos itens não influenciará no significado da lista. Ela é apenas uma maneira de organizar os itens sem uma classificação de importâcia.

Para criar uma unordered list, vamos usar a tag <ul> para delimitar a lista e a tag <li> para criar um dos seus itens internos. 

            
    <ul>
        <li>item 1</li>
        <li>item 2</li>
        <li>item 3</li>
        <li>item 4</li>
        <li>item 5</li>
    <ul>

O marcador padrão é o circulo preto preenhechido (circle), mas existe a opção de configurar a propriedade type da tag <ul> com os seguintes nalores:

  • disc - padrão. Um circulo preto preenhechido
  • circle - Um circulo preto sem preenhechido
  • square - Um pequeno quadrado preto preenhechido

 Exemplo

Lista de compras

  • pão
  • leite
  • tomate
  • carne
  • sabão
  • refrigerante
  • manteiga

 Misturando as coisas

Podemos também criar listas mistas, configurando dentro de outras listas. Veja o exemplo aseguir:

        
        <ol>
            <li>item 1</li>
            <li>item 2</li>
            <ol type="a">
                <li>item 2.a</li>
                <li>item 2.b</li>
                <li>item 2.c</li>
        <ol>
            <li>item 3</li>
            <li>item 4</li>
            <li>item 5</li>
            <ol>

O resultado visual do código que vimos anteriormente será semelhante ao que temos a seguir:

  1. item 1
  2. item 2
    1. item 2.a
    2. item 2.b
    3. item 2.c
  3. item 3
  4. item 4
  5. item 5

Note que os itens da lista interna (A, B, C, ...) está deslocado para a direita em relação ao item 2, do qual essa sub-lista é filha.

DICA!

Além de aninhar listas ordenadas, podemos juntar listas com <ol> e vice-versa. As listas internas senpre terão deslocamento interno para direita.

 Listas de Definições


É como se fosse um dicionário, temos os termos e as suas descrições. É uma lista sem dimarcadores, mas bem útil em alguns casos.

Toda lista de Definições está dentro de uma tag <dl> </dl> (definition list). Cada termo é um <dt> (definition term) e cada descrição é um <dd> (definition description). Assim como os itens da lista, essas duas últimas tags possuem fechamento opcional, segundo a referência oficial da HTML5.

 exemplo:

HTML
Linguagem de marcação ultilizada para criar conteúdo de sites.
CSS
Linguagem de marcação usada para a especificação de estilos em sites.
JavaScript
Linguagen de programação para criar interatividades em sites.
