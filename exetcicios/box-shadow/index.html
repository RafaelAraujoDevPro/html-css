Sombra de caixa

O shadow-box é uma propriedade do CSS, e é utilizado para adicionar efeitos de sombras em volta do elemento. Podemos especificar mais de um efeito, os separando com vírgula.

A sequência dos valores de shadow-box, aplicando em shorthand, representam os seguintes valores:

  1. Deslocamento horizontal (h-offset, ou eixo X): quando a sombra vai apontar na horizontal da direita para esquerda com valores positivo. Valores negativos apontam da esquerda para direita.
  2. Deslocamento vertical (v-offset, ou eixo Y): quando a sombra apontará de cima para baixo com valores positivos. Valores negativos apontam de baixo para cima
  3. Embaçamento (blur): aplica o quanto a sombra se espalhará pelo o fundo da caixa do elemento.
  4. Cor (color): aplica a cor da sombra. É possível controlar a opácidade da cor.
