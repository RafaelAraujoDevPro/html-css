Propriedade SVG

O nome SVG vem de Scalable Vector Graphics (Gráficos Vetoriais Escaláveis). Este é basicamente um formato de imagem vetorial suportado pelos navegadores e que pode ser animado ou modificado por CSS e JavaScript. O elemento SVG é um contêiner que define um novo sistema de coordenadas e de veiwport. Ele é usado como elemento externo de documentos SVG, mas também pode ser usado para incorporar um fragmento SVG dentro de um ducumento SVG ou HTML.

As vantagens de usar SVG

O SVG possui várias vantagens, como:

  • Poderm ser animados por CSS e modificados por JavaScript.
  • Por serem imagens vetoriais, podemos alterar o tamanho livrimente sem perda de qualidade da imagem.
  • Podem ser embutidas no HTML. diminuíndo a quantidade de requisições.
  • Não é necessário um super editor de imagem para trabalhar com SVG. Algumas coisas simples podem ser editadas com o próprio editor de códigos ou usar até mesmo as páginas da web.
  • Possui várias funcionalidades que ainda são impossíveis ou muito complexas de se fazer com HTML e CSS, nos permitindo criar formas e efeitos mais complexos para nossas páginas web.
  • Como pode ser integrado com CSS, podemos usar Media Queries para criar imagens responsivas.

Sintaxe

Podemos criar uma arquivo .svg e depois chamar em nosso HTML usando a tag <img>, mas também pode-se, dentro do seu próprio HTML, criar uma tag <svg> e começar a escrever o código dentro dele. vamos começar com uma exemplo usando o vetor <rect> para criarmos um retângulo:

                    
                        <svg>
                             <rect width="200" height="100" fill-"#D95B29" x="30" y="30"
   stroke="#A9F0E1" stroke-width="5"></rect>
                        </svg>

Resultado

Vamos entender o que está declarado no exemplo:

  • <rect> declara que queremos criar um retângulo
  • width e height servem para declararmos a largura e altura do elemento
  • fill serve para declararmos a cor do preenchimento do elemento.
  • x e y servem para indicar o posicionamento do elemento svg em relação ao contêiner em que está contido.
  • stroke nos permite declarar a cor da linha do contorno
  • stroke-width declara a grossura da linha de contorno do elemento.

Outro vetor muito comum usado na tag <svg> é o <circle>, que serve para criar um círculo

                    
                        <svg>
                            <circle r="30" cx="50" cy="40" fill="#D95B29" stroke="##A9F0E1" stroke-width="5">
                        </svg>

Resultado

Como estilizar svg com CSS

Os elementos svg tem uma grande vantagem sobre os outros tipos de arquivos de imagens porque pode ser estilizado de maneira muito mais poderoza quando usamos as CSS. A tag svg compartilha algumas propriedades com o CSS mas possui muitas outras propriedades exclusivas para estilizar seus elementos de imagem.

Compartilhado com CSS Somente SVG
font, font-family, font-size, font-size- adjust, font-stretch, font-style, font- variant, font-weight, direction, letter- spacing, text-decoration, unincode-bidi, word-spacing, visibility, text-rendering writing-mode, clip-path, mask-opacity, filter, pointer-events, image-rendering, clip, color, cursor, display, overflow. clip-rule, flood-color, flood-opacity, stop-opacity, kerning, tech-anchor, color-profile, color-rendering, fill, fill-opacity, fill-rule, marker, marker- end, marker-mid, marker-start, stroke, stroke-width, stop-color, lighting-color, enable-background, dominant-baseline, color-interpolation-filters, color- interpolation, glyph-orientation- horizontal, glyph-orientation-vertical, shape-rendering, baseline-shift, alignment-baseline, stroke-miterlimit, stroke-linejoin, stroke-linecap, stroke- dashoffset, stroke-dasharray, stroke- opacity.

Existem varias maneiras para se declarar essas propriedades na tag svg, e assim como no CSS, existe um peso de especificidade para elas que é igual a especificidade do CSS.

Estilo inline

A forma de estilo inline é aplicada a tag style diretamente na tag do elemento <svg>

                    
                        <svg style="width: 300px; height: 300px;" viewBox="0 0 300 300">
                               <polygon
                               style="fill: #D95B29; stroke: #A9F0E1; stroke-width: 5;"
                               points="100,10 40,198 190,78 10,78 160,198"/>
                        </svg>

Estilo dentro do svg (presentation Attibutes)

Essa outra maneira de estilizar o svg declaramos os atributos e seus valores diretamente na tag svg sem a tag style. A tag style será declarada como elemento filho de svg.

                    
                        <svg width="300" height="300" viewBox="0 0 300 300">
                                <style>
                                       polygon {
                                            fill: red;
                                            stroke: yellow;
                                            stroke-width: 5;
                                    }
                                </style>
                                <polygon
                                            points="100,10 40,198 190,78 10,78 160,198"/>
                        </svg>

Estilo fora do svg

Outra maneira de estilizar svg seria declará uma tag style no documento HTML e dentro dela um seletor com os atributos de svg e depois referenciar o seletor no elemento contido na tag svg. Seguindo nas mesma ideia, também podemos referenciar um seletor que vem de um link de CSS eXterno em relação ao documento do HTML (rel="stylesheet" href="/style/estilo.css").

                    
                        <style>
                             .estrela_svg {
                                    fill: #D95B29;
                                    stroke:#A9F0E1;
                                    stroke-width: 5;
                                    }
                         <style/>
                        .......
                        <svg>
                            <polygon class="estrela_svg"
                              points="100,10 40,198 190,78 10,78 160,198">
                        </svg>

Viewport e ViewBox

Os elementos svg são manipulados dentro de um espaço específico para eles que é a viewport da tag <svg>. já o viewBox é o atributo que define a posição e a dimensão destes elementos que estão no espaço da viewport svg.

O valor do atributo viewBox é uma lista de quatro números: min-x, min-y, width e height. Os números declarados para min-x e min-y representam as coordenadas superiores esquerdas da viewport (apartir de x="0" y="0"). os números para width e height representam suas dimensões. Esses números, que são separados por espaço em branco e/ou vírgula, especificam um retângulo no espaço do usuário que é mapeado para os limites da viewport estabelicida para o elemento SVG associado (não a viewport do navegador).

                    
                        <svg class="cor_fundo" width="300" height="100"
 viewBox="0 0 400 150" style="border: 1px solid blue;">
                                <circle r="50" cx="200" cy="75"/>
                        </svg>

Formas Básicas do SVG

A linguagem SVG possui seis formas básicas de constuir gráficos que também são chamadas de objetos primitivos da SVG, e são usadas isoladamente para criação de gráficos simples ou em conjunto para desenvolvimento de gráficos avançados. São eles:

  • <rect />
  • <circle />
  • <ellipse />
  • <line />
  • <polyline />
  • <polygon />

<rect />

Os atributos xml para o elemento <rect> destinados á criação de um retângulo são:

  • x → define a coordenada horizontal do canto superior esquerdo do retângulo;
  • y → define a coordenada vertical do canto superior esquerdo do retângulo;
  • width → define a largura do retângulo;
  • height → define a altura do retângulo;
  • rx → define o raio de curvatura segundo o eixo x para arredondamento dos cantos do retângulo;
  • ry → define o raio de curvatura segundo o eixo dos y para arredondamento dos cantos do retângulo.
                    
                        <svg class="cor_fundo" width="300" height="100"
 viewBox="0 0 400 150" style="border: 1px solid #A9F0E1">
                                <rect x="125" y="37.5" width="150" height="75" rx="10" ry="10"/>
                        </svg>

<circle>

Os atributos XML para o elemento <circle> destinados à criação de um círculo são:

  • r → define o raio do círculo;
  • cx → define a coordenada horizontal do centro do círculo;
  • cy → define a coordenada vertical do centro do círculo;
                    
                        <svg class="cor_fundo" width="300" height="100"
viewBox="0 0 400 150" style="border: 1px solid #A9F0E1">
                                <circle r="70" cx="200" cy="75" fill="#D95B29" stroke="#A9F0E1"
      stroke-width="4"/>
                        </svg>

<ellipse>

os atributos XML para o elemento <ellipse> destinados à criação de uma elipse são:

  • cx → define a coordenada horizontal do centro da elipse;
  • cy → define a coordenada vertical do centro da elipse;
  • rx → define o raio horizontal da elipse;
  • ry → define o raio vertical da elepse;
                    
                        <svg class="cor_fundo" width="300" height="100"
 viewBox="0 0 400 150" style="border: 1px solid blue">
                                <ellipse cx="200" cy="75" rx="90" ry="60" fill="green"
     stroke="yellow" stroke-width="4"/>
                        </svg>

<line>

Os atributos XML para o elemento <line> destinados à criação de uma linha reta são:

  • x1,y1 → definem a coordenada do ponto inicial da linha reta;
  • x2,y2 → definem a coordenada do ponto final da linha reta;
  • stroke-width → define a espessura da linha reta;
  • stroke → define a cor da linha reta;
                    
                        <svg class="cor_fundo" width="300" height="100"
 viewBox="0 0 400 150" style="border: 1px solid blue">
                                  <line x1="10" y1="10" x2="100" y2="120" stroke="#000" stroke-width="4"/>
                        </svg>

<polyline>

O atributo XML para o elemento <polyline> destinados à criação de polilinhas é o atributo points. O valor deste atributo é uma lista de coordenadas separadas por espaço em branco. A polilinha é uma sequência de segmentos de reta unindo os pontos definidos pelas coordenadas

                    
                        <svg class="cor_fundo" width="300" height="100"
      viewBox="0 0 300 100" style="border: 1px solid #A9F0E1">
                                <polyline fill="transparent" stroke="#176583" stroke-width="4"
                                points="0,20 30,20 30,40 60,40 60,60 90,60 90,80 120,80 "/>
                        </svg>

<polygon>

O atributo XML para o elemento <polygon> destinados à criação de polígonos é o atributo points. O valor deste atributo é uma lista de coordenadas separadas por espaço em branco. O polígono é uma sequência de segmentos de reta unindo os pontos definidos pelas coordenadas, formando uma superfície fechada.

Observe o código a seguir destinado a criar um hexágono (polígono de seis lados) passando pelos pontos de coordenadas "125,10 175,10 200,50 175,90 125,90 100,50 125,10".

                    
                        <svg class="cor_fundo" width="300" height="100"
 viewBox="0 0 300 100" style="border: 1px solid blue">
                                <polygon fill="#D95B29" stroke="#A9F0E1" stroke-width="4"
                                points="125,10 175,10 200,50 175,90 125,90 100,50 125,10"/>
                        </svg>
