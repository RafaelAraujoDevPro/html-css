Pseudo-Classes

Uma pseudo-classe de CSS é uma palavra-chave especificada selecionada para um elemento especial selecionado. Por exemplo, :hover pode ser usado para alterar um botão quado o usuário passar o cursor sobre ele.

Sintaxe

                
                    seletor:pseudo-classe {
                        propriedade: valor;
                    }

As pseudo-classes permitem que você não aplique um estilo a um elemento em relação ao conteúdo da árvore do documento, mas também em relação aos fatores externos como o histórico de navegação (:visited por exemplo), o status do conteúdo (como :checked em certos elementos de um formulário), ou à posição do mouse (como :houver, que permite saber se o mouse está sobre um elemento ou não).

Podemos usar as pseudo-classes para aplicar estilos à um determinado elemento, assim como qualquer seletor, porérm possuem uma diferença signidicativa com relação aos demais seletores que é a seleção de um elemento HTML de acordo com o seu estado, ou então à seleção de uma parte específica de um elemento HTML ao invés dele como um todo.

Geralmente uma pseudo-classe está associada à algum seletor definido, através de seletores de tag, id, class e atributo. A pseudo-classe inicia com :(dois pontos) sem (espaço), logo após o seletor, e em seguida o nome da dela.

Exemplo

                
                    button:hover {
                        color: blue;
                    }
                    #lista:ferst-child {
                        font-style: italic;
                    }
                    .btnActive:active {
                    }
                    input[type = text]:focus {
                        /*formatação*/
                    }

As pseudo-classe mais usudas normalmente serão listadas a seguir:

Estados

Um estado da pseudo-classe acontece quando o usuário desencadeia uma ação. Uma "ação" em CSS pode ser também um "não ação" como no casode um link que ainda não tenha sido visitado.

:link

A pseudoclasse :link é usada para estilizar elementos ancorados por links, que ainda não foram viditados. É recomendado que ela seja a primeira a ser declarada entre todas as pseudo-classes destinadas à estilizar os estados de links. A ordem de declaração dos quatro estados é: :link, :visited, :houver, :active

:visited

A pseudo-classe :visited é usada para estilizar links visitados deve ser a segunda a ser declarada (depois de :link)

:houver

:houver é usada para estilizar elementos para que quando o usuário passe o cursor sobre ele mude o estado do estilo deste elemento para o estilo definido na pseudo-classe.

:active

A pseudo-classe :active é usada para estilizar elementos para que quando o usuário torne este elemento "ativo" através de um click por exemplo, mude o estado de estilo dele para o estilo definido na pseudo-classe.

:focus

:focus é parecida com :houver, porém é mais usada em caixa de formulário, para que quando, ela receba fóco através do cursor passando sobre, mude para o estilo definido na pseudo-classe.

Pseudo-classes estrururais

Pseudo-Classes estruturais casam com porções da árvore do documento (ou DOM) que não podem ser casadas com uso de outros tipos de seletores ou elementos de combinações de seletores.

:first-child

A estilização de :first-child casará com o elemento do primeiro filho de seu elemento pai.

                    
                        HTML
                        <ul>
                            <li>Este é o primeiro elemento</li>
                            <li>Este é o segundo elemento</li>
                            <li>Este é o último elemento</li>
                        </ul>
                    
                
                    
                        CSS
                        li:first-child {
                            color: orange;
                        }

Resultado

  • Este é o primeireo elemento
  • Este é o segundo elemento
  • Este é o último elemento

:first-of-type

A pseudo-classe :first-of-type casa com o primeiro elemento de um determinado tipo que seja filho de seu elemento pai;

HTML

                    
                        HTML
                        <dl>
                               <dt>Este é o primeiro elemento</dt>
                               <dd>Este é o segundo elemento. <span>Tipo especifico</span></dd>
                               <dd>Este é o último elemento</dd>
                        </dl>
                    
                
                    
                        CSS
                        ul:first-of-type {
                               color: red;
                        }

Resultado

Este é o primeireo elemento
Este é o segundo elemento, o tipo específicado
Este é o último elemento

:last-child

A pseudo-classe :last-child casa com o último elemento de seu elemento pai. No exemplo a seguir o último li será estilizado através da pseudo-classe.

                    
                        HTML
                        <ul>
                            <li>Este é o primeiro elemento</li>
                            <li>Este é o segundo elemento</li>
                            <li>Este é o último elemento</li>
                        </ul>
                    
                
                    
                        CSS
                        li:last-child {
                            color: red;
                        }

Resultado

  • Este é o primeireo elemento
  • Este é o segundo elemento
  • Este é o último elemento

:last-of-type

A pseudo-classe :last-of-type casa com o último elemento de um determinado tipo que seja filho de seu elemento pai. No exemplo a seguir o último li com span será estilizado.

                    
                        HTML
                        <dl>
                           <dt>Este é o primeiro elemento</dt>
                           <dd>Este é o segundo elemento. </dd>
                           <dd>Este é o último elemento, <span>Tipo especifico</span></dd>
                    </dl>
                    
                
                    
                        CSS
                        li:last-of-type {
                            color: darkgreen;
                        }

Resultado

Este é o primeireo elemento
Este é o segundo elemento, o tipo específicado
Este é o último elemento

:not

A pseudo-classe :not é também como pseudo-classe de negação. Ela aceita um argumento (basicamente um outro seletor) escrito entre sinais de parênteses. O argumento poderá ser, também, uma pseudo-classe encadeada porém não pode conter outro seletor :not.

No exemplo a seguir os elementos de li estarão estilizados através do seletor de tag ul exceto a li que receberá uma classe específica.

                    
                        HTML
                        <ul>
                               <li>Este é o primeiro elemento</li>
                               <li clas="destaque">Este é o segundo elemento, porem estará negado a estilização.</li>
                               <li>Este é o terceiro elemento</li>
                               <li>Este é o último elemento</li>
                        </ul>
                    
                
                    
                        CSS
                        li:not(.destaque) {
                               background-color: yellow;
                               color: blue;
                        }

Resultado

  • Este é o primeireo elemento
  • Este é o segundo elemento, porem estará negado a estilização.
  • Este é o terceiro elemento
  • Este é o último elemento

:nht-child

A pseudo-classe nht-child casa com um ou mais elementos dependendo da ordem em que eles aparecem na marcação HTML. Esta pseudo-classe é uma das mais versáteis e robustas da CSS. :nth adimite um argumento que consiste em uma fórmula a ser inserida entre parênteses. A fórmula poderá ser um simples número inteiro, algo do tipo an+b ou as palavras odd (impar) ou even (par).

A fórmula an+b:

  • a letra a é um número inteiro;
  • a letra n é o literal n (ou seja, a própria letra n);
  • o sinal + é o operador matemático que pode ser soma (+) ou subtração (-);
  • a letra b é um número inteiro que deverá estar presente na fórmula somente se for usado o operador matemático.

O exemplo a seguir mostrará um lista do alfabeto grego de acordo com a estrutura HTML.

                    
                        HTML
                        <ul>
                            <li>Alfa</li>
                            <li>Beta</li>
                            <li>Gamma</li>
                            <li>Delta</li>
                            <li>Epdilon</li>
                            <li>Zeta</li>
                            <li>Eta</li>
                            <li>Theta</li>
                            <li>Iota</li>
                            <li>Kappa</li>
                        </ul>

Vamos estilizar na cor laranja (orange) o terceiro filho, ou seja, a palavra "Gamma" será estilizada.

                        
                            CSS
                            ul :nht-child(3) {
                                color: orange;
                            }

:nth-last-child

A pseudo-classe :nth-last-child é idêntica à pseudo-classe :nht-child exceto que o casamento se dá contando os elementos de trás para frente.

Vamos estilizar na cor laranja (orange) o terceiro filho contando de trás pra frente, ou seja, a palavra "Theta" será estilizada.

                        
                            CSS
                            ul :nht-last-child(3) {
                                color: orange;
                            }

E assim sucessivamente repetindo os exemplos de :nht-child de trás pra frente.

Resultado

  • Gamma
  • Delta

  • Zeta
  • Eta
  • Theta
  • Iota
  • Kappa

nth-of-type

A pseudo-classe :nth-of-type é idêntico à pseudo-classe :nht-child. A principal diferença é que se trata de um seletor mais específico, pois casa com elementos filhos de determinado tipo.

No exemplo a seguir o segundo paragrafo contido no elemento pai <article> será estilizado na cor de fonte laranja.

                        
                            <div>
                                <h2>Aqui há um titulo</h2>
                                <p>Lorem ipsum dolor sit.</p>
                                <a href="#"><img src="images/foto.jpeg" alt="foto"></a>
                                <p>Este elemento será na cor laranja.</p>
                            </div>
                        
                    
                        
                            p:nth-of-type(2) {
                                color: orange;
                            }

Resultado

Aqui há um título

Lorem ipsum dolor sit.

Este elemento será na cor laranja.

:only-child

A pseudo-classe :only-child casa com o elemento filho único dentro de um elemento pai.

No exemplo a seguir o primeiro ul tem um único elemento filho cújo o texto será estilizado na cor laranja (orange). O segundo elemento ul tem varios elementos filhos li, neste caso nenhum deles seram estilizados quando se usa a pseudo-classe :only-child

                        
                            <ul>
                                <li>Este será cor laraja.</li>
                            </ul>
                            <ul>
                                <li>Lorem ipsum dolor sit.</li>
                                <li>Lorem ipsum dolor sit.</li>
                            </ul>
                        
                    
                        
                            ul li:only-child {
                                color: orange;
                            }

Resultado

    • Este será cor laraja
  • Este é o segundo elemento
  • Este é o último elemento

only-of-type

A pseudo-classe :only-of-type casa com elementos que não têm irmãos do seu tipo. Ele é idêntico ao elemento :only-child exceto pelo fato de que casa com um determinado tipo de elemento, o que faz com que o seletor seja mais específico.

                        
                            <div>
                                <h2>Aqui há um titulo</h2>
                                <p>Lorem ipsum dolor sit.</p>
                                <p>Este <span>elemento</span> será na cor laranja.</p>
                                <h2>Aqui há um titulo</h2>
                                <p>Lorem ipsum dolor sit.</p>
                            </div>
                        
                    
                        
                                p :only-of-type {
                                color: orange;
                            }

Resultado

Aqui há um titulo

Lorem ipsum dolor sit.

Este elemento será na cor laranja.

:target

A pseudo-classe :target pode ser usado para estilizar um elemento-destino específico, que está vinculado à um link de navegação da página, através da tag <a>. Para isso o href indentificador de fragmentação deve ser o nome do seletor id, que deve estar dentro de qualquer tipo de elemento no documento HTML, como atributo.

Depois então no CSS declaramos o elemento que contém o id, adicionamos a pseudo pseudo-classe :target e a formatação do estilo. Exemplo de Sintaxe: section:target {*/formatação*/} .

No exemplo a seguir, o elemento que será destino do elemento de navegação, receberá estilização através da pseudo-classe :target quando a rolagem automática acontecer.

                        
                            HTML
                            <nav>
                                <ul>
                                    <li><a href="#exemplo1">e.elemento 1</a></li>
                                    <li><a href="#exemplo2">e.elemento 2</a></li>
                                    <li><a href="#exemplo3">e.elemento 3</a></li>
                                </ul>
                            </nav>
                            <article id="#exemplo1">
                            <h3>elemento 1</h3>
                                <p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Cum, praesentium.<p/>
                            </article>
                            <article id="#exemplo2">
                            <h3>elemento 2</h3>
                                <p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Cum, praesentium.<p/>
                            </article>
                            <article id="#exemplo3">
                            <h3>elemento 3</h3>
                                <p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Cum, praesentium.<p/>
                            </article>

Neste exemplo há um lista conteúdos da página para o usuário navegar. Quando isso acontecer o conteúdo será mostrado com a estilização da pseudo-classe :target

                        
                            CSS
                            article:target {
                                outline: 3px dotted red;
                            }
