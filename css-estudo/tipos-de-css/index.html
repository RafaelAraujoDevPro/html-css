CSS

O que é CSS?

CSS (Cascading Style sheets) é uma linguagem que usamos pra estilizar um documento HTML. As CSS descreve como os elementos HTML devem ser exibidos. As CSS economizam muito trabalho quando precisamos desenvolver sites com um número alto de páginas. Ele pode controlar o layout de várias páginas da web de uma só vez.

As CSS podem ser usadas de três maneiras: pode ser através do Style Inline, do Style Local/interno ou Style Esterno.

Style inline.

O Style inline é usudo dentro das tags do HTML como atributos. Sempre é inserido na tag de abertura onde podemos definir o estilo para o conteúdo da tag como cor,tamanho e tipo de fonte do texto, cor de fundo do e etc...

Essa forma de inserir CSS para o site é a menos usual pois demanda mais trabalho, ainda mais se tivermos muitas páginas neste site para desenvolver folhas de estilos. Geralmente aproveitamos o Style inline estilizar um conteúdo pontual em uma única página apenas.

Style Local/interno

Usamos Style Local logo no início do nosso documento ainda dentro da tag <head>. Desta forma os estilos declarados ficam restritos em uma única página, não sendo possível a sua reutilização em outras páginas, ao menos que copiemos e colemos nas outras, porém demada um pouco mais de trabalho.

Exemplo

        
        <html>
          <head>
                <style type="text/css">
                            h2 {color:#000;}
                </style>
          </head>
        <body>

Style Externo

Este tipo de uso de CSS em documentos HTML é o mais indicado e recomendado. Neste tipo, os códigos CSS ficam em um documento externo ao documento HTML, separando assim as camadas de estrutura e apresentação do projeto WEB. Existe duas formas de incorporar folhas de estilos CSS externa ao documento WEB. Veja a seguir:

1. Linkadas

Para o tipo de estilo CSS externo linkado, usamos o elemento link para incorpora o documento CSS externo ao documento HTML. O elemento link deve ser usado dentro da seção <head> do documento HTML.

Vamos ver um exemplo da utilização do elemento link para incorporar uma folha de estilo CSS ao documento HTML. Neste exemplo, imagine que o documento CSS esteja dentro de uma pasta chamada "css" e o seu arquivo HTML, "index.html" esteja na raiz,como na ilustração:

        
        <head>
             <link rel="stylesheet" type="text/css"
              media="screen" href="css/estilos.css">
       </head>

Partindo do princípio que o documento CSS estja nomeado como "estilos.css" e que esteja dento de uma pasta chamada "css", o atibuto href deve dicar exatamente com esta informação: "css/estilo.css", que é a forma com que o navegador entenderá onde se encontra o arquivo CSS a ser linkado.

O estilo linkado é a maneira mais usual para estilizar sites pois assim as folhas de estilos carregam mais rapido.

2. Importadas

O tipo de estilo das CSS importadas funciona igualmente ao tipo de estilo linkadas. O que difere aqui é somente a forma com que isto é feito, uma vez que, neste caso, é usada a diretiva @import dentro do elemento style, na seção <head> do documento HTML, como pode ser visto no exemplo:

    
        <head>
            <style type=“text/css”>
                      @import url("css/estilos.css") screen,
                       projection;
            </style>
        <head>
Se o desenvolvedor pretede ganhar velocidade, não é recomendado o uso de estilos importadas, pois o @import demora mais para carregar a folha de estilo.
