Tag <table>

O que é uma tabela HTML?

Uma tabela é uma representação de dados organizados em linhas e colunas. Realmente, é mais como uma planilha. Em HTML, com a ajuda de tabelas, podemos organizar dados como imagens, texto, links e assim por diante em linhas e colunas de células.

Para criar uma tabela em HTML vamos precisar da tag <table> que funcionará como um container principal da tabela para agrupar os elementos filho dela.

As tags filhas de <table>

aparecem a seguir:

  • <tr>→ representa linhas.
  • <td>→ usado para criar células de dados.
  • <th>→ usado para adinionar títulos de tabela
  • <caption>→ usado para inserir legendas.
  • <thead>→ adiciona cabeçalho separado à tabela.
  • <tbody>→ mostra o corpo principal da tabela.
  • <tfoot>→ cria um rodapé separado para a tabela.

Exemplo

Veja a sintáxe de tabela HTML

                    
                        <table>
                         <tr>
                            <td>cell 1</td>
                            <td>cell 2</td>
                            <td>cell 3</td>
                         </tr>
                         <tr>
                            <td>cell 4</td>
                            <td>cell 5</td>
                            <td>cell 6</td>
                         </tr>
                        </table>

Corresponde assim:

cell 1 cell 2 cell 3
cell 4 cell 5 cell 6

Como adicionar um título de tabela HTML

O <th> é usado para adicionar títulos ás tabelas. Em designs básicos, o cabeçalho da tabela sempre ficará na linha superior, o que significa que teremos o <th> declarando em nossa primeira linha da tabela seguido pelos dados reais na tabela.

Exemplo

                    
                        <table>
                            <tr>
                                <th>First Name</th>
                                <th>Last Name</th>
                                <th>Email Address</th>
                            </tr>
                            <tr>
                                <td>Rafael</td>
                                <td>>Araújo</td>
                                <td>table@gmail.com</td>
                            </tr>
                            <tr>
                                <td>Fulano</td>
                                <td>Beltrano</td>
                                <td>developer@gmail.com</td>
                            </tr>
                        </table>
Name Last Name Email Address
Rafael Araújo table@gmail.com
Fulano Beltrano developer@gmail.com
A partir do exemplo acima, podemos dizer qual coluna contém quais informações. Isso é possível com o uso de <th>

Adicionar legenda em tabelas

O principal uso de adicionar uma legenda à tabela é fornecer uma descrição sobre os dados representados na tabela. A legenda pode ser colocada na parte superior ou inferior e, por padrão, estará sempre centralizada. A tag <caption> é usada para inserir legendas em tabelas.

Veja um exemplo da Sintaxe de <caption>

                    
                        <table>

Atributo scope em tabelas HTML

O atributo scope é usado para definir se um título específico é destinado à uma coluna, linha ou grupo de ambos. O objetivo principal de scope é mostrar os dados de destino para que o usuário não precise confiar em suposições.

O atributo é declarado na célula de título e assim a tag <th> recebe valores col, row, colgroup e rowgroup. Os valores col e row indicaram que a célula de título está fornecendo informações para as linhas ou colunas, respectamente.

Veja um exemplo de sintaxe da propriedade scope.

                    
                        <table>
                        <tr>
                        <th scope="value"></th>
                        </tr>
                        </table>

Exemplo:

                    
                        <table>
                            <tr>
                                <th></th>
                                <th scope="col">Semester</th>
                                <th scope="col">Grade</th>
                            </tr>
                            <tr>
                                <td>1</td>
                                <td>Jan - April</td>
                                <td>Credit</td>
                            </tr>
                            <tr>
                                <td>2</td>
                                <td>May - August</td>
                                <td>Pass</td>
                            </tr>
                            <tr>
                                <td>2</td>
                                <td>September - December</td>
                                <td>Distinction</td>
                            </tr>
                            </table>
Semester Grade
1 Jan - April Credit
2 May - August Pass
2 September - December Distinction
O que o atributo scope faz é mostrar se uma célula de cabeçalho pertence à coluna, à linha ou a um grupo de ambas. Neste caso, os cabeçalhos pertencem à coluna porque é o que definimos no código.

Divisão de células em tabelas HTML

Quando uma tabela esta estendida sobre duas ou mais linhas ou colunas chamamos isso de extensão celular. Para isso usamos dois atributos na célula: colspan para expansão horizontal e rowspan para expansão vertical. Para os dois atributos inserimos o número maior que zero, que são a quantidade de células que queremos abranger.

Exemplo

                    
                        <table>
                            <tr>
                                <th>Name</th>
                                <th>Subject</th>
                                <th>Marks</th>
                            </tr>
                            <tr>
                                <td rowspan="2">Hillary</td>
                                <td>advanced Web</td>

                            </tr>

                                <td>75</td>

                            <tr>
                                <td></td>

Cabeçalho, corpo e rodapé em tabelas HTML

Assim como em um site ou qualquer outo documento tem três seções principais: o cabeçalho, o corpo e o rodapé. Também acontece o mesmo com tabela HTML. para indentificar essas seões usamos as tags <thead>, <tbody> e <tfoot>

  • <thead> → fornece um cabeçalho separado da tabela.
  • <tbody> → contém o conteúdo principal da tabela.
  • <tfoot> → cria um rodapé separado para a tabela.

Vamos ver um exemplo com o uso de <thead>, <tbody> e <tfoot>

Exemplo:

                
                    <table>
                        <thead>
                            <tr>
                                <th colspan="2">October</th>
                                <th colspan="2">Nobember</th>
                            </tr>
                        </thead>
                        <tbody>
                            <tr>
                                <td>Sales</td>
                                <td>Profit</td>
                                <td>Sales</td>
                                <td>Profit</td>
                            </tr>
                            <tr>
                                <td>200,00</td>
                                <td>50,00</td>
                                <td>300,00</td>
                                <td>70,00</td>
                            </tr>
                        </tbody>
                            <tr>
                                <th colspan="4">November was more produstive</th>
                            </tr>
                        </tfoot>
                    </table>
October November
Sales Profit Sales Profit
200,00 50,00 300,00 70,00
November was more produstive
No exemplo acima, o cabeçalho é representado pelo nome dos meses, a parte dos números de vendas e lucro representa o corpo da tabela e, por fim, a parte com o comando representa o rodapé da tabela.
