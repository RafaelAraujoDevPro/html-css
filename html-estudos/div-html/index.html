HTML div: estilizando páginas com a tag <div>

Estudo de artigo.

O que é e para que serve a tag <div> no HTML?

A tag <div>, que correspode a abreviatura de Division, é um elemento HTML utilizado para dividir e agrupar os itens que serão exibidos em uma página. Por isso, ele funciona como em container,que pode receber diferentes tipos de conteúdo e orgnizá-los de várias maneiras.

Para organizar as tags divs em uma página HTML, é preciso utilizar os estilos CSS. São eles que ajudam a definir as características de cada container, como o tamanho, o posicionamento e muitas outras difinições para que seja possível realizar a organização desejada dos elementos inseridos na <div>.

Apesar de ser muito utilizada para a construção de layouts, a tag <div> não interfere no SEO → Search Engine Optimization ← da página, pois não faz parte do grupo de elementos semânticos do HTML, que são as tags que indicam o que cada parte do conteúdo representa, como os elmentos <header>, <section>, entre outras. Portanto, os mecanismos de buscas não utilizam essa tag para classificar o conteúdo do site.

Como usar a tag <div>?

A tag <div> é utilizada para separar e agrupar conteúdos. Para isso, todos os elementos que pertencem a ela devem ser declarados entre a tag de início <div> e a tag de fechamento </div>. Assim, os estilos aplicados sobre a <div> refletirão nos itens inseridos nela.

Para que o elemento <div> cumpra seu papel de separar e agrupar outros elementos, ele deve trabalhar em conjunto com os estilos CSS. Isso porque essas definições são responsávis por proporcionar características específicas à ele.

Por padrão, a tag <div> contém a característica de inserir uma quebra de linha antes de sua declação, já que ela tem a função de separar elementos na página. Isso acontece, pois a maioria dos navegadores atribui o estilo "display: block" à tag <div>. Entretanto, é possível alterar o posicionamento por meio da aplicação de estilos.

Os principais atributos da tag HTML <div>

Assim como outros elementos HTML, a tag <div> suporta os atributos e eventos globais da linguagem. Dessa forma, é possível atribuir características por meio de estilos CSS e ralizar alterações por meio de códigos JavaScript. Confira os principais atributos do elemento <div>

Estilizado o elemento <div>: Class e ID

O atributo class é declarado com um ponto "." da denominação da classe, por exemplo, ".teste" e referenciado no atributo "class" do elemento. Ele pode ser aplicado em qualquer elemento e diversas vezes em uma página.

<div class="teste"></div>

Já o conteúdo do atributo id só pode ser atribuido em um único elemento na página, pois ele representa a sua identificação, que não pode se repetir. A declaração de id em CSS é feita usando o caractere "#" antes do nome do id. por exemplo: "#idTeste".

<div id="idTeste"></div>

A diferença entre estes dois atributos está em relação a individualização, ou seja, quando queremos que um <div> tenha características únicas, utilizamos o atributo id. Já quado queremos repetir a aparência em diversas divs, utilizamos o atributo class. Veja um exemplo:

Diferença entre <div> e <span>

É comun duvidas em relação a utilização das tags <div> e <apan>. Isso porque as duas servem para delimitar um determinado conteúdo e aplicar o estilo CSS atribuidos a elas. A principal diferença entre elas é o atributo display, que por padrão é definido como "block" na tag <div> e "inline" na tag <span>.

O display block significa que existe uma quebra de linha no início e no encerramento da tag. Veja o algorítimo abaixo:

                
                    <DOCTYPE html>
                    <html lang="en">
                    <head>
                    <meta charset="UTF-8">
                    <meta name="viewport" content="width=device-width,
                        initial-scale=1.0">
                    <title>Tags div e span</title>
                    <style type="text/css">
                        .divs{
                            background-color: darkcyan;
                            border: 1px solid #000;
                        }
                        .span1 {
                            background-color: burlywood;
                            border: 2px solid red;
                        }
                        .span2 {
                            background-color:chocolate;
                            border: 2px solid red;
                        }
                    </style>
                    </head>
                    <body>
                    <h1>Diferença entre as tags div e span</h1>
                    <div class="divs">Primeira div</div>
                    <div class="divs">Segunda div</div>
                    <span class="span1">Primeira span</span>
                    <span class="span2">Segunda span</span>
                    <div class="divs">Outra div</div>
                    </body>
                    </html>

Ao depurar esse código em um navegador, perceba que a primeira <div> será iniciada em uma nova linha, após o encerramento da tag <h2>, assim como a segunda <div>. Já o conteúdo das duas spans ocupa a mensma linha, enquanto a próxima <div> será iniciada na linha sequinte.

Além disso, o elemento <div> ocupa toda a linha caso o seu tamanho não seja especificado, mesmo que seu conteúdo termine antes. Já a tag <apan> ocupa somente o espaço necessátio para a exibição dos elementos dentro dela.

Como fazer o posicionamento do elemento <div>: CSS position

Por padrão, o elemento <div> tem a largura da página, Entretanto, podemos aplicar qualquer tamanho a ele e, dessa forma, podicioná-lo de diferentes maneiras na página, como fixá-lo no topo, posicionar as divs uma ao lado da outra e muito mais. Isso é feito por meio da propriedade position, que determina como em elemento será posicionado na página e aceita os seguintes atributos:

  • ralative: em que o posicionamento é baseado nas definições das propriedades top e bottom para o deslocamento horizontal e left e right em relação horizontal. O deslocamento do elemento é feito em relação ao posicionamento em que ocupa na página;
  • absolute: em que o posicionamento é definido pelas peopriedades top, bottom,left e right e em relação ao topo da página;
  • fixed: em que o elemento dica na mesma posição, mesmo com a rolagem da página;
  • sticky: funciona como um adesivo, ou seja, em um primeiro monento o posicionamento é relativo. entretanto, ao rolar a página, esse atributo faz com que a <div> permaneça no posicionamento indicado pelas outras popriedades auxiliares.
